# Fra Mor til Jord

Praktiske verktøy for livets store steg – fra det nye livet tar form til det gamle avsluttes.

## Struktur

```
/
├── index.html            ← Felles forside
├── barnehage/            ← Verktøy for barnehage og tidlig barndom
│   ├── index.html        ← Status på barnehagesøknad
│   ├── finner.html       ← Finn barnehageplass i Oslo
│   ├── opptak.html       ← Forstå opptaksprosessen
│   ├── permisjonsplanlegger.html
│   └── statusguide.html
└── dod/                  ← Planlegging rundt livets slutt
    └── research/         ← Research-dokumenter (grunnlag for fremtidige sider)
```

## Filosofi

Verktøyene er laget for å hjelpe vanlige mennesker i vanskelige og forvirrende livssituasjoner. Ingen data lagres. Ingen pålogging. Bare praktisk hjelp.

## Teknologi

Ren HTML/CSS/JavaScript – ingen byggsteg, ingen avhengigheter. Åpne filen i nettleseren og det virker.
